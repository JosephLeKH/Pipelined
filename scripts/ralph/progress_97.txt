**Implementation:** Auto-trigger fit score generation in the background when an email creates a new application, similar to `_trigger_interview_prep`.

  - Calls Gemini with career coach system prompt for 0-100 fit scoring
  - Parses JSON response and validates score is integer 0-100
  - Persists fit_score, fit_score_reason, fit_score_at to applications collection
  - Handles timeout and parse errors gracefully

- Follows pattern established by `_trigger_interview_prep()` for background task execution

**Architecture:**
- Shared fit score logic in fit_score.py allows both router and email service to call it
- Lazy import in email_integration/service.py prevents circular import with applications.interview_prep
- Respects existing fire-and-forget pattern for background tasks
- Timezone-aware datetime (datetime.now(timezone.utc)) to avoid deprecation warnings
